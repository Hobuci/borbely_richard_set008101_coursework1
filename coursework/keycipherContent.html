Subtitution Key Cipher


Takes an input text and a key, shifts each character of the message by the location of the key's characters in the alphabet. ex. Message ABC decoded with the key 'B' would shift each letter by 1, because 'B' is the second character of the alphabet (alphabet[1]); BCD
A key can be any length, but if it's shorter than the message, the key is going to be repeated to match the message length.
Can only process letters of the alphabet separated by spaces.
